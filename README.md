# Science Tokyo LMS 課題通知 Bot

Science Tokyo LMS の課題情報をTampermonkeyからCloudflare D1へ同期し、Discordで課題一覧の表示と期限通知を行う個人用Bot。

## 期限通知

期限までの残り時間に応じて、現在の区分だけを通知します。24時間未満は1時間区分、2日未満は「2日」、3日未満は「3日」、3日以上7日以内は「7日以内」です。この優先順位により、2日未満の課題へ「7日以内」を重ねて通知しません。

## Discordの緊急度表示

`/課題` の応答と期限通知は、締切までの残り時間に応じて色と絵文字が変わります。課題一覧は締切順に並び、残り時間を太字の独立した行で表示します。

| 残り時間 | 表示 | Embed色 |
|---|---|---:|
| 期限超過 | ⛔ 期限超過 | `0x4B4B4B` |
| 1時間以内 | 🚨 超緊急 | `0xFF2D2D` |
| 3時間以内 | 🔴 緊急 | `0xFF4D4F` |
| 24時間以内 | 🟠 注意 | `0xFF8C00` |
| 3日以内 | 🟡 近い | `0xFFD60A` |
| 7日以内 | 🟢 通常 | `0x34C759` |
| 7日より先 | 🔵 先の予定 | `0x3B82F6` |

24時間以内の通知だけが、設定済みのユーザーまたはロールをメンションします。`allowed_mentions` ではそのIDだけを許可し、`@everyone` と `@here` は許可しません。`/同期状態` には24時間・3日・7日以内の課題件数が追加され、最終同期から24時間以上経過している場合は警告を表示します。

変更をCloudflareへ反映するには、型検査とテストの後にデプロイします。

```cmd
npm run typecheck
npm test
npx wrangler deploy --dry-run
npm run deploy
```

今回の表示変更ではD1マイグレーション、Secretの再登録、Discordコマンドの再登録、Tampermonkeyの更新は不要です。
